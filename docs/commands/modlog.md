# modlog

> Generated from the shared Aegis command catalog.

## Summary

Set the moderation case log channel.

## Description

Stores the text channel used for moderation actions such as kicks, bans, mutes, and strike-based punishments.

## Syntax

- `^modlog [#channel|off]`

## Access

Manage Server permission

## Aliases

None

## Examples

- `^modlog #mod-logs`
- `^modlog off`

# antinuke disable

> Generated from the shared Aegis command catalog.

## Summary

Disable anti-nuke enforcement for the server.

## Description

Stops live anti-nuke enforcement until it is re-enabled. This does not erase thresholds, trust entries, or incident history.

## Syntax

- `^antinuke disable`

## Access

Server owner or anti-nuke trusted entry

## Aliases

None

## Examples

- `^antinuke disable`

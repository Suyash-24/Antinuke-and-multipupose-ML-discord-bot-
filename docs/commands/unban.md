# unban

> Generated from the shared Aegis command catalog.

## Summary

Remove an active ban from one or more users.

## Description

Unbans the selected users, clears scheduled temporary-ban expirations, and records the action in the case log.

## Syntax

- `^unban <users...> [reason]`

## Access

Ban Members permission or configured mod role

## Aliases

None

## Examples

- `^unban 123456789012345678 appeal accepted`
- `^unban @user mistaken identity`

# modrole

> Generated from the shared Aegis command catalog.

## Summary

Set or clear the extra moderator role recognized by Aegis.

## Description

Lets you designate one role as a moderation override so members with that role can use moderation commands even if they do not hold the raw Discord permission.

## Syntax

- `^modrole [role]`

## Access

Manage Server permission

## Aliases

None

## Examples

- `^modrole @Staff`
- `^modrole`

# setup muted

> Generated from the shared Aegis command catalog.

## Summary

Create or register the muted role used by Aegis.

## Description

Creates the `Aegis Muted` role when needed, applies restrictive overwrites across channels where possible, and stores the role for future mute actions.

## Syntax

- `^setup muted`

## Access

Manage Server permission

## Aliases

None

## Examples

- `^setup muted`

## Notes

- Run this before using `^mute` if your server has not configured a muted role yet.

# automod antiraid

> Generated from the shared Aegis command catalog.

## Summary

Configure the join-burst trigger for automatic raid mode.

## Description

Turns automatic raid mode on or off, or sets a custom joins-per-seconds threshold that enables raid mode during suspicious join bursts.

## Syntax

- `^automod antiraid <on|off|joins/seconds>`

## Access

Manage Server permission

## Aliases

`automod autoraidmode`, `automod autoraid`, `automod autoantiraid`

## Examples

- `^automod antiraid on`
- `^automod antiraid 10/8`
- `^automod autoraid off`

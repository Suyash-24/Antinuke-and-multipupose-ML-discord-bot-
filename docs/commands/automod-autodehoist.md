# automod autodehoist

> Generated from the shared Aegis command catalog.

## Summary

Configure automatic nickname dehoisting.

## Description

When enabled, Aegis watches for nicknames that begin with a specific hoist character and normalizes them automatically.

## Syntax

- `^automod autodehoist <character|off>`

## Access

Manage Server permission

## Aliases

`automod dehoist`

## Examples

- `^automod autodehoist !`
- `^automod dehoist off`

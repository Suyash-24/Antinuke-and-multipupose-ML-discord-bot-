# automod resolvelinks

> Generated from the shared Aegis command catalog.

## Summary

Enable or disable redirect-link resolution.

## Description

When enabled, Aegis resolves redirect-style URLs so invite and referral checks can look through certain wrappers.

## Syntax

- `^automod resolvelinks <on|off>`

## Access

Manage Server permission

## Aliases

None

## Examples

- `^automod resolvelinks on`
- `^automod resolvelinks off`

- None

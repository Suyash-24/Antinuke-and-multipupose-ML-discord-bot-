# automod antireferral

> Generated from the shared Aegis command catalog.

## Summary

Set the referral-link punishment.

## Description

Controls how Aegis reacts to suspicious referral or tracking-link spam such as affiliate or redirect-heavy links.

## Syntax

- `^automod antireferral <strikes|delete|off>`

## Access

Manage Server permission

## Aliases

None

## Examples

- `^automod antireferral 1`
- `^automod antireferral delete`

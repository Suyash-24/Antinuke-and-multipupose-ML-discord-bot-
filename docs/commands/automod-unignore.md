# automod unignore

> Generated from the shared Aegis command catalog.

## Summary

Remove a role or channel from the AutoMod ignore list.

## Description

Re-enables normal AutoMod checks for the selected role or channel.

## Syntax

- `^automod unignore <role|channel>`

## Access

Manage Server permission

## Aliases

None

## Examples

- `^automod unignore #staff-chat`
- `^automod unignore @Trusted`

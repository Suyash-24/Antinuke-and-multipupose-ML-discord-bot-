# pardon

> Generated from the shared Aegis command catalog.

## Summary

Manually remove strikes from one or more members.

## Description

Reduces a member's active strike total without deleting case history so moderators can correct or forgive past moderation decisions.

## Syntax

- `^pardon [count] <users...> <reason>`

## Access

Moderate Members permission or configured mod role

## Aliases

None

## Examples

- `^pardon @user appeal accepted`
- `^pardon 2 @user corrected mistake`

# Settings

Server configuration commands for logs, setup, and moderation defaults.

## Commands

### [`setup`](setup.md)

Show available setup actions for first-time server configuration.

- **Syntax:** `^setup`
- **Access:** Manage Server permission

### [`setup muted`](setup-muted.md)

Create or register the muted role used by Aegis.

- **Syntax:** `^setup muted`
- **Access:** Manage Server permission

### [`modrole`](modrole.md)

Set or clear the extra moderator role recognized by Aegis.

- **Syntax:** `^modrole [role]`
- **Access:** Manage Server permission

### [`modlog`](modlog.md)

Set the moderation case log channel.

- **Syntax:** `^modlog [#channel|off]`
- **Access:** Manage Server permission

### [`messagelog`](messagelog.md)

Set the message edit/delete log channel.

- **Syntax:** `^messagelog [#channel|off]`
- **Access:** Manage Server permission

### [`serverlog`](serverlog.md)

Set the server event log channel.

- **Syntax:** `^serverlog [#channel|off]`
- **Access:** Manage Server permission

### [`voicelog`](voicelog.md)

Set the voice event log channel.

- **Syntax:** `^voicelog [#channel|off]`
- **Access:** Manage Server permission

### [`settings`](settings.md)

Show the current Aegis configuration for the server.

- **Syntax:** `^settings`
- **Access:** Manage Server permission

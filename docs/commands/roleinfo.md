# roleinfo

> Generated from the shared Aegis command catalog.

## Summary

Inspect a role's basic details and permissions.

## Description

Shows the role ID, member count, position, mentionability, creation time, and a preview of enabled permissions.

## Syntax

- `^roleinfo <role>`

## Access

Everyone

## Examples

- `^roleinfo Staff`

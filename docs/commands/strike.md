# strike

> Generated from the shared Aegis command catalog.

## Summary

Manually add strikes to one or more members.

## Description

Adds strikes to the selected members and immediately checks the configured punishment ladder for any threshold they just reached.

## Syntax

- `^strike [count] <users...> <reason>`

## Access

Moderate Members permission or configured mod role

## Aliases

None

## Examples

- `^strike @user advertising`
- `^strike 2 @user repeat offense`

# automod antiinvite

> Generated from the shared Aegis command catalog.

## Summary

Set the invite filter punishment.

## Description

Controls how Aegis reacts when users post Discord invite links. You can disable it, delete only, or assign strikes.

## Syntax

- `^automod antiinvite <strikes|delete|off>`

## Access

Manage Server permission

## Aliases

None

## Examples

- `^automod antiinvite 2`
- `^automod antiinvite delete`
- `^automod antiinvite off`

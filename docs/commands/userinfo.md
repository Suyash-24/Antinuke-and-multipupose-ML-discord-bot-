# userinfo

> Generated from the shared Aegis command catalog.

## Summary

Inspect a member's identity and server profile.

## Description

Shows the user ID, display name, top role, join date, and account creation time. If no user is provided, Aegis shows your own profile.

## Syntax

- `^userinfo`
- `^userinfo <member>`

## Access

Everyone

## Aliases

None

## Examples

- `^userinfo`
- `^userinfo @Suyash`

# antinuke protect

> Generated from the shared Aegis command catalog.

## Summary

Turn a specific anti-nuke event family on or off.

## Description

Enables or disables detection for one anti-nuke event type without changing the saved threshold numbers.

## Syntax

- `^antinuke protect <event> <on|off>`

## Access

Server owner or anti-nuke trusted entry

## Aliases

None

## Examples

- `^antinuke protect role_update on`
- `^antinuke protect guild_update off`

- None

# voicekick

> Generated from the shared Aegis command catalog.

## Summary

Disconnect one or more members from voice.

## Description

Moves the selected members out of voice by disconnecting them, then records a voice moderation case.

## Syntax

- `^voicekick <users...> [reason]`

## Access

Move Members permission or configured mod role

## Aliases

None

## Examples

- `^voicekick @user loud mic spam`
- `^voicekick @user1 @user2 voice raid`

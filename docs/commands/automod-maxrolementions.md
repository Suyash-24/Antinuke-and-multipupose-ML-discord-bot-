# automod maxrolementions

> Generated from the shared Aegis command catalog.

## Summary

Set the threshold for role mentions in one message.

## Description

Triggers AutoMod when a message mentions too many roles at once.

## Syntax

- `^automod maxrolementions <count|off>`

## Access

Manage Server permission

## Aliases

None

## Examples

- `^automod maxrolementions 2`
- `^automod maxrolementions off`

# automod ignore

> Generated from the shared Aegis command catalog.

## Summary

Exclude a role or channel from AutoMod enforcement.

## Description

Adds a role or guild channel to the ignore list so AutoMod skips checks there.

## Syntax

- `^automod ignore <role|channel>`

## Access

Manage Server permission

## Aliases

None

## Examples

- `^automod ignore #staff-chat`
- `^automod ignore @Trusted`

- None

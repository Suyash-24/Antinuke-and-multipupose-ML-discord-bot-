# automod maxlines

> Generated from the shared Aegis command catalog.

## Summary

Set the line-count threshold for multi-line spam.

## Description

Flags messages that exceed the configured number of lines, which is helpful against pasted wall spam.

## Syntax

- `^automod maxlines <count|off>`

## Access

Manage Server permission

## Aliases

None

## Examples

- `^automod maxlines 12`
- `^automod maxlines off`

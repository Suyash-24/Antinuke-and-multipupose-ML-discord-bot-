# serverlog

> Generated from the shared Aegis command catalog.

## Summary

Set the server event log channel.

## Description

Stores the channel used for joins, leaves, nickname changes, role updates, channel changes, and broader guild event logs.

## Syntax

- `^serverlog [#channel|off]`

## Access

Manage Server permission

## Aliases

None

## Examples

- `^serverlog #server-logs`
- `^serverlog off`

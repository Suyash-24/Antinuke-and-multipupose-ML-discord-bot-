# automod maxmentions

> Generated from the shared Aegis command catalog.

## Summary

Set the mention threshold for regular user mentions.

## Description

Triggers AutoMod when a single message exceeds the configured user-mention count.

## Syntax

- `^automod maxmentions <count|off>`

## Access

Manage Server permission

## Aliases

None

## Examples

- `^automod maxmentions 5`
- `^automod maxmentions off`

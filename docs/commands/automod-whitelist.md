# automod whitelist

> Generated from the shared Aegis command catalog.

## Summary

Manage the invite whitelist for allowed guild IDs.

## Description

Lets you add, remove, or show guild IDs that bypass the Anti Invite rule. This whitelist is for destination guilds, not for user accounts.

## Syntax

- `^automod whitelist <add|remove|show> [guild-id...]`

## Access

Manage Server permission

## Aliases

`automod whitelistinvites`

## Examples

- `^automod whitelist show`
- `^automod whitelist add 123456789012345678`

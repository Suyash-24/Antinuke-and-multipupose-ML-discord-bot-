# check

> Generated from the shared Aegis command catalog.

## Summary

Inspect a member's moderation state in one panel.

## Description

Shows current strikes, mute state, remaining mute time, ban state, remaining ban time, and the best available ban reason.

## Syntax

- `^check`
- `^check <user>`

## Access

Moderate Members permission or configured mod role

## Aliases

None

## Examples

- `^check`
- `^check @user`

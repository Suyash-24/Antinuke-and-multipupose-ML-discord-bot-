# General

Everyday info and discovery commands for normal users and moderators.

## Commands

### [`help`](help.md)

Show command categories or detailed help for a category or command.

- **Syntax:** `^help`
- **Access:** Everyone

### [`about`](about.md)

Show a quick snapshot of the bot, uptime, and command count.

- **Syntax:** `^about`
- **Access:** Everyone

### [`invite`](invite.md)

Get the invite link and required permissions for Aegis.

- **Syntax:** `^invite`
- **Access:** Everyone

### [`ping`](ping.md)

Check the bot's WebSocket latency.

- **Syntax:** `^ping`
- **Access:** Everyone

### [`roleinfo`](roleinfo.md)

Inspect a role's basic details and permissions.

- **Syntax:** `^roleinfo <role>`
- **Access:** Everyone

### [`serverinfo`](serverinfo.md)

View a live overview of the current server.

- **Syntax:** `^serverinfo`
- **Access:** Everyone

### [`userinfo`](userinfo.md)

Inspect a member's identity and server profile.

- **Syntax:** `^userinfo`
- **Access:** Everyone
